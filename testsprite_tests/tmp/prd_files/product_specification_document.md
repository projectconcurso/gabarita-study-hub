# Product Specification Document

## 1. Product Overview
The product is a digital platform designed to support users in studying and preparing for exams through interactive simulations, performance tracking, and a gamified progression system. The platform will provide question-based assessments and reward users with points (XP) and virtual currency (Gabaritos) based on performance.

## 2. Purpose
The main purpose of the product is to improve learning efficiency by combining practice testing with gamification elements. It aims to:
- Increase user engagement
- Provide measurable progress tracking
- Simulate real exam environments
- Offer flexible study options

## 3. Target Audience
- Students preparing for public exams (concursos)
- Users seeking structured study routines
- Individuals who prefer practice-based learning

## 4. Core Features

### 4.1 Simulated Exams
- Users can select the number of questions per simulation
- Questions are categorized by subject and difficulty
- Timed and untimed modes available

### 4.2 Gabaritos System (Virtual Currency)
- Users spend Gabaritos to start simulations
- Cost structure based on number of questions:
  - 1–20 questions: 5 Gabaritos
  - 21–40 questions: 10 Gabaritos
  - 41–60 questions: 15 Gabaritos
  - 61+ questions: scalable cost

### 4.3 Experience Points (XP) and Leveling
- Users earn XP based on performance
- Every 10 XP increases user level
- Levels unlock features and benefits

### 4.4 Performance Metrics
- Accuracy rate (taxa de acerto)
- Historical performance tracking
- Progress by subject and topic

### 4.5 Plans and Monetization
- Free Plan:
  - Limited Gabaritos
  - Advertisements enabled
- Monthly Plan:
  - 500 Gabaritos per month
  - Reduced or no ads

### 4.6 User Dashboard
- Overview of performance
- Current XP and level
- Gabaritos balance
- Recent activity

## 5. User Flow
1. User creates an account
2. User receives initial Gabaritos
3. User selects a simulation
4. Gabaritos are deducted
5. User completes the simulation
6. System calculates score and XP
7. Results are displayed and stored

## 6. Technical Requirements
- Web-based platform (responsive)
- Secure authentication system
- Scalable backend for handling simulations
- Database for user data and performance

## 7. Compliance and Data Protection
- Must comply with LGPD (Lei Geral de Proteção de Dados)
- User data must be securely stored and encrypted
- Clear consent for data usage

## 8. Future Enhancements
- Mobile application
- Social ranking and leaderboards
- AI-based study recommendations
- Custom question creation

## 9. Success Metrics
- User retention rate
- Daily active users
- Average simulations per user
- Conversion rate to paid plans

## 10. Conclusion
This product aims to modernize the study experience by combining structured assessments with gamification, providing users with an engaging and efficient way to prepare for exams.
